Note: 
====
Time series 10, 11 and -15 can target all groups in a fleet by specifying group 0. You can still
provide individual fleet+group time series of these types, which get presedence:

	Has time series value fleet + group? 
	Yes: 
		Use that
	No:
		Has time series value fleet + 0?
		Yes: 
			Use that
		No:
			Do not apply time series value
